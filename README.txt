Project Title:
VITB_RELATIVE_GRADE_CALCULATOR

Overview:
This project calculates the grades based on relative grading method. The user uploads an Excel file which has students marks in Google Colab. The program processes the marks and generates a new Excel file with grade columns.

Motivation: The purpose behind the research and the data collected.

Data Description: A detailed explanation of the data files, including their format, contents, and how they were created.

Features:
Upload Excel marks file
Automatic calculation of mean and standard deviation 
Relative grading for S, A, B, C, D, E
Hard fail: F for marks less than 40
Final Excel file downloaded with grades

System Requirements:
• Google Colab 
• Microsoft Excel 

Learning Outcomes
This project demonstrates:

Conditional statements (if-elif-else)
Loops (while loop)

Technologies Used:
Python
Panda

How to use:
1. Open the Google Colab notebook.
2. Run the code
3. Upload the excel file containing marks.
4. Run all code cells.
5.The final Excel file containing grades will be downloaded in seconds after running all the codes.
